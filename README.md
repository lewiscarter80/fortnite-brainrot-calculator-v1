## Description

This web application offers a dedicated calculator for the Fight the BRAINROT mode in Fortnite. Players can enter their character attributes and instantly receive a combat power rating, assisting with decisions on gear and upgrades. The tool accounts for multiple in-game factors to produce accurate results without manual calculation.

Version 1.0 introduces mutation support, letting you see how different mutations influence your overall power. A reverse calculation mode also lets you work backward from a target power level to identify the necessary stats. This release focuses on greater accuracy and broader configuration support for both casual and competitive play.
